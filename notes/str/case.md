# CASE

## `title()`

```python
print('ada lovelace'.title())  # Ada Lovelace
```

## `lower()`, `upper()`

```python
name = 'Ada Lovelace'
print(name.lower())  # ada lovelace
print(name.upper())  # ADA LOVELACE
```

## `capitalize()`

```python
print('hello, how are you?'.capitalize())  # Hello, how are you?
```
