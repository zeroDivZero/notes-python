# VARIABLE AND DATA TYPES

Dynamic type. Type implicitly set when assigned.

## Common Types

Boolean: `bool`
Number: `int`, `float` (no double)
String: `str`
Sequence: `list`, `tuple`
Mapping: `dict`
Set: `set`
Binary: `bytes`
`None`: `NoneType`

## Boolean

```python
is_game_over = False
```

## Number

```python
num_lives = 5
percent_health = 0.5
```

## String

```python
player_name = "Peter" # single quotes also acceptable
```

## Determining Type

```python
print(type(player_name)) # <class 'str'>
```
