# `print()`

```python
print('Hello, world!')
```

## `end`

What to print at end. Default: `\n` (line feed).

```python
print('Cheese', end=' ')
print('Burger')
# prints 'Cheese Burger'
```

## `sep`

If printing multiple strings, default separator is space. Can customize:

```python
print('cats', 'dogs', 'mice', sep=', ')  # cats, dogs, mice
```
