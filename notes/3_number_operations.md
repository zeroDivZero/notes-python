# NUMBER OPERATIONS

## Arithmetic

```python
2+2 # 4
```

```python
7.0/2 # 3.5
```

```python
8/5 # 1.6; fractions aren't lost when dividing integers
```

```python
7//-3 # -3; integer division
```

```python
3**4 # 81; exponential: 3 to 4th power
```

## Conversion

To float:

```python
float(3) # 3.0
```

To int:

```python
int(5.0) # 5
```
